# vertx-helloworld
simplest example of vertx 3.x

# how to build
gradle clean build

# how to run 

## command line
`java -jar ./build/libs/vertx-helloworld-1.0-SNAPSHOT.jar`

This will deploy an verticle listening on port 8080
